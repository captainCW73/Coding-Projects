# To-Do List Application

This project is a simple command-line to-do list application that allows users to manage their tasks with checkboxes. It provides functionalities to add, remove, and toggle tasks, as well as format them into a readable list.

## Project Structure

```
todo-list-app
├── src
│   ├── main.py          # Entry point of the application
│   ├── formatter.py     # Contains the Formatter class for task formatting
│   ├── checklist.py     # Contains the CheckList class for task management
│   └── utils.py        # Utility functions for task management
├── requirements.txt     # Lists the dependencies required for the project
└── README.md            # Documentation for the project
```

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd todo-list-app
   ```

2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

To run the application, execute the following command:
```
python src/main.py
```

Follow the on-screen instructions to manage your tasks. You can add tasks, remove them, and toggle their completion status.

## Features

- Add tasks to your to-do list
- Remove tasks from your list
- Toggle the completion status of tasks
- View tasks in a formatted list with checkboxes
